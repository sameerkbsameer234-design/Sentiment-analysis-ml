# Sentiment-analysis-ml
Machine Learning project for emotion classification using NLP
# Sentiment Analysis using Machine Learning

## 📌 Objective
This project aims to classify emotions in text using machine learning models.

## 🧠 Techniques Used
- Text Preprocessing (Cleaning, Tokenization, Stopword Removal)
- TF-IDF Feature Extraction
- Machine Learning Models:
  - Naive Bayes
  - Support Vector Machine (SVM)

## 📊 Dataset
Dataset contains text samples labeled with emotions.

## ⚙️ Models Used

### 1. Naive Bayes
- Fast and efficient
- Works well for text classification

### 2. Support Vector Machine (SVM)
- High accuracy
- Handles high-dimensional data effectively

## 📈 Results
- Compared models using Accuracy and F1-score
- SVM performed better in most cases

## 🚀 How to Run
1. Open the notebook in Google Colab
2. Upload dataset
3. Run all cells

## 📌 Conclusion
SVM is more suitable for emotion classification due to better performance on text data.
